# evang_planner
For scraping event info and planning routes.

## Setup - local python scripts (Unix)
`cd server`
`python3 -m venv evang_pyenv`
`source evang_pyenv/bin/activate`
`python3 -m pip install --upgrade pip`
`python3 -m pip install -r requirements.txt`

## Use - local python scripts (Unix)
`source evang_pyenv/bin/activate`
`python3 scrape_ohio_festivals.py`
